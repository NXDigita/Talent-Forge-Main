# TALENT FORGE — Full-Stack Web Application Prompt
## AI-First | Blockchain-Enabled | Gamified Talent Marketplace

---

## 🧠 PROJECT OVERVIEW

Build a full-stack, responsive web application called **Talent Forge** — a next-generation AI-powered Talent Marketplace Platform for India. The platform connects skill-verified engineering graduates with startups, MSMEs, and enterprises through gamified assessments, real-world simulations, and blockchain-backed credentials.

The app must have a **dark-professional design system** with a navy + electric blue + amber accent palette. It should feel like a cross between **Linear.app** (clean SaaS), **HackerRank** (assessment UX), and **Duolingo** (gamification energy).

---

## 🎨 DESIGN SYSTEM

### Color Palette
- Primary Background: `#0F172A` (slate-900)
- Card Background: `#1E293B` (slate-800)
- Border: `#334155` (slate-700)
- Primary Blue: `#2563EB`
- Electric Blue Accent: `#38BDF8`
- Amber Accent: `#F59E0B`
- Success Green: `#22C55E`
- Text Primary: `#F8FAFC`
- Text Secondary: `#94A3B8`
- Gradient Hero: `linear-gradient(135deg, #0F172A 0%, #1E3A5F 50%, #0F172A 100%)`

### Typography
- Font: Inter (Google Fonts)
- Hero Heading: 64px, bold, white with gradient text on key words
- Section Heading: 36px, bold
- Body: 16px, slate-300
- Labels/Tags: 12px, uppercase, letter-spacing 0.1em

### UI Style Rules
- Rounded corners: `border-radius: 16px` on cards, `8px` on buttons
- Cards: glassmorphism effect — `background: rgba(30,41,59,0.8)`, `backdrop-filter: blur(12px)`, `border: 1px solid rgba(255,255,255,0.08)`
- Hover effects: subtle glow — `box-shadow: 0 0 24px rgba(37,99,235,0.3)`
- Buttons: gradient fill `linear-gradient(90deg, #2563EB, #38BDF8)` with white text
- Micro-animations: fade-in on scroll, slide-up for cards, pulse on badges
- All icons: Lucide React icon set

## 🔧 TECHNICAL REQUIREMENTS

### Stack
- **Framework:** React 18 + TypeScript
- **Styling:** Tailwind CSS (dark mode first) + custom CSS variables
- **Routing:** React Router v6
- **State:** React Context + useState/useReducer (no Redux needed)
- **Charts:** Recharts (for dashboard analytics)
- **Icons:** Lucide React
- **Animations:** Framer Motion (scroll reveal, counters, transitions)
- **Forms:** React Hook Form
- **Notifications:** React Hot Toast

### Component Architecture
Create reusable components:
- `<Navbar />` — sticky, glassmorphism
- `<HeroSection />` — animated headline
- `<StatCounter />` — animated number count-up on scroll
- `<FeatureCard />` — glassmorphism card
- `<ProjectCard />` — marketplace listing
- `<TalentCard />` — candidate profile card
- `<PricingCard />` — pricing tier
- `<BadgeChip />` — domain / tier tag
- `<ProgressBar />` — XP / score bar
- `<TFESGauge />` — circular score gauge (SVG)
- `<DashboardSidebar />` — collapsible nav
- `<AssessmentQuestion />` — question card
- `<MissionCard />` — gamified task card

### Interactivity Requirements
- All dashboard charts use Recharts with real-looking mock data
- Assessment page: clicking answers highlights selection, advances question, updates gauge
- Marketplace filters actually filter the shown cards (client-side)
- Earnings calculator recomputes on every slider/dropdown change
- ROI calculator updates instantly
- Stats strip: numbers count up with Framer Motion when scrolled into view
- Pricing toggle: Monthly / Annual (annual shows 33% savings)
- FAQ accordions open/close smoothly
- All CTA buttons show a toast notification: "🚀 Feature coming soon! Join the waitlist."
- Mobile hamburger menu with slide-in drawer

### Responsive Design
- Mobile-first breakpoints: sm (640px), md (768px), lg (1024px), xl (1280px)
- Navbar collapses to hamburger on mobile
- Dashboard sidebar collapses to icon-only on tablet, hidden drawer on mobile
- All grids: 1 col mobile → 2 col tablet → 3 col desktop
- Hero: stacked on mobile, side-by-side on desktop
